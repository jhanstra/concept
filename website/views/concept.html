Concept

A modern, flexible CSS framework for designers and creators.

Alpha 0.1.5

Welcome to Concept

Concept is still in alpha and not ready for production use. However, please feel free to look around and check stuff out, download the source and/or contribute! The goal of this framework is to combine the best parts of some of the most popular CSS frameworks, and learn a lot about each of them along the way. Bootstrap, Foundation, Semantic UI, and Skeleton provide the inspiration for this framework. Each of these frameworks has many strengths, but also many shortcomings and annoyances. Concept will be some sort of hybrid of these, with a modern, sleek design. It's built by Jared for his web projects, but you can use it too, I guess =)

Why use Concept?

Concept brings together some of the best features of other CSS frameworks and introduces a few tricks of its own. It's not too big, not too small. Customizable, but not too customizable. Powerful, yet easy to use.

Next-gen

Concept uses cutting-edge CSS3 features and follows modern design trends and techniques. It is designed for power users and doesn't support older browsers (<IE 9).

Beautiful

Every pixel in Concept's classes has been meticulously designed. No matter what kind of web thing you want to make, Concept will make it look beautiful.

Modular

Concept allows you to build custom-styled elements by combining CSS classes. Every class does only what it needs to, and no more. Need a small, rounded success button with lowercase letters that presses down on hover?

No problem

Flexible

Customization with Sass variables is not only possible, it's encouraged. Concept makes it easy to completely change the appearance of your website with just a handful of variables.

Prefixed

Always know where your styles are coming from. Concept's classes are prefixed with so you always know which styles are coming from the Concept framework and which are coming from elsewhere.

33K

Concept currently measures in at only 33kb. It's not too big, and not too small. And you can easily customize it for an even smaller footprint, using only what you need.
