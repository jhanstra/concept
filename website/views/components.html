Header

Add a header to the top of your page with .header. You can easily change the height of the header with $header-height.





Types of Headers

Add .header-primary to give your page a 500px header. This is the default size. Use .header-secondary for secondary pages, which has a default height of 250px. Concept also comes with full-height headers, which consume the full window height. For this, add .header-full. This site uses full-height headers.





Background Images

You can give your headers a background-image with .header-bg-image-1 and by editing the $header-background-image-1 variable. Images are automatically centered and will fill the containing space for all screen sizes.

Horizontal Navigation

.horizontal-nav places a horizontal navbar at the top of its containing element. Simply add a list inside this nav and populate with navigation links.





Vertical Navigation

Use .vertical-nav for a vertical sidebar. It is best placed at the same level as your .container section. You can put the sidebar on the left or right.

Labels

Horizontal Rule

Add .hr-full for a horizontal rule that spans the full width of its containing element. Add .hr-half for one that is 50% of the width, centered. Use .hr-mini for a mini <hr>, by default 50px.
