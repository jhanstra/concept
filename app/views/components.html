Components

Concept provides dozens of components that you can plug and play in your website or application. Choose which components to include in the concept.scss file and add classes to your html to use the styles for each component.

Header & Navigation

Add a header to the top of your page with .c-header. You can easily change the height of the header with $header-height. Add a vertical sidebar with .c-vertical-nav or horizontal navigation with .c-horizontal-nav.

Header

The <header> element introduces the content of your page, and is often set apart visually from the rest of the page. Add .c-header-primary to give your page a jumbo (500px height by default) header. Or use .c-header-full to add a full-height (the height of the viewport) header to your page. This site uses full-height headers.

Concept also offers .c-header-secondary for secondary pages, which has a default height of 250px.

You can give your headers a background-image with .c-header-bg-image-1 and by editing the $header-background-image-1 variable. Images are automatically centered and will fill the containing space for all screen sizes. Pretty neat.

Miscellaneous

Horizontal Rule

Add .c-hr-full for a horizontal rule that spans the full width of its containing element. Add .c-hr-half for one that is 50% of the width, centered. Use .c-hr-mini for a mini <hr>, by default 50px.
