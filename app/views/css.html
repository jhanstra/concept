Typography

All headers h1 through h6 have a default font of Helvetica Neue. All p tags have a default font of Merrieweather Sans. You can easily change these fonts by changing the variables $header-font and $paragraph-font, respectively, in variables.scss

Grid

Concept uses a responsive 12-column grid that can be changed from 1 to 50 columns with the $number-of-columns variable. The grid does not give any default padding or margin to columns or rows. Rather, these styles are added using special classes.

Buttons

Use a variety of classes to build buttons. Start with .c-btn and add classes as you wish. Choose between 7 different sizes: extra small, small, normal, large, jumbo, fixed-width, or full-width. Choose between flat buttons, raised buttons, outline buttons, rectangle buttons, or clicky buttons.
